Is iFakku free?

iFakku is 100% free and ad-free. I have no plans to use iFakku for monetization purposes.

I plan to release the source code for iFakku at some point. I don't know when this will be. You should just know that the code is an absolute mess.

I found a bug/have a suggestion, how do I contact you about it?

The best way to get in touch with me is through the Fakku website itself. Just post your suggestion or bug onto the official iFakku thread under Forums > FAKKU Developers. I'm also on the Fakku IRC a lot under the handle TYN, so you can chat with me there as well.

Any usage recommendations when using the app?

Alright, so a lot of pages are reused, so there's a lot of checking code. My only suggestion is to view Downloaded manga from the Downloaded tabs since the images are loaded internally, and thus, much faster.

What are the "Fakku Forums" and "Official iFakku Thread" buttons on the User page?

I just put those there to make it easier to get to the forums. They're simply web views displaying the respective pages. Obviously, if you think I'm out to steal your password because I somehow rigged the web view (I didn't), don't use it. I lurk the forums a lot.

Version 3.0? What happened to Version 2.0?

So I was programming version 2.0, but then I was super preoccupied with real life, so development stopped for many months. A lot of the code in iFakku 2.0 was also very experimental, so when I tried to start development again, I would just get too confused and stop. iFakku 3.0 is a fresh version of iFakku that refined the code in iFakku 2.0 and I feel it warrants a new version.
